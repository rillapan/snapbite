Saya punya sebuah proyek web full-stack lokal (Node.js backend + Vanilla JS/HTML website frontend). Saya ingin menginisialisasi Git di dalam proyek ini dan melakukan push pertama kali ke repositori GitHub saya.

Berikut adalah detail repositori remote saya:
URL Repo: https://github.com/rillapan/snapbite.git
Branch utama: main

Tolong buatkan dan eksekusi langkah-langkah Git di terminal untuk saya secara otomatis atau pandu saya langkah demi langkah. Pastikan:
1. File sensitif seperti `.env`, folder `.claude/`, node_modules/, dan file kredensial lainnya dimasukkan ke dalam file `.gitignore` terlebih dahulu agar tidak ikut ter-track atau bocor ke GitHub.
2. Lakukan inisialisasi git, buat file .gitignore jika belum ada, add semua file yang aman, dan buat commit pertama dengan pesan "initial commit: setup project architecture".
3. Hubungkan ke origin remote di atas dan lakukan push ke branch main.

Berikan perintah terminalnya yang bersih dan siap dieksekusi di Windows PowerShell/CMD.
